Compute the statistics of converted Eclipse files

This tool will compute the statistics the user wishes to compute and store these in the configured database (can be changed in the settings). Determining which statistics will be calculated can be performed by selecting from the statistical options. For now, these are: 'all', 'p_p_amplitude' and 'auc'. In the future there might be additional statistics that can be computed.
After clicking the run button you will have to fill out an experiment form. Once it is filled in correctly, windows containing the IONM signals will appear. In these, you can deselect signals or change up the ranges of which the statistics will get computed.

Please select one or more files you wish to compute the statistic of.
