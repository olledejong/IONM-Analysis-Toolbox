Obtain file characteristics

Using this tool, you will be able to retrieve some basic information about the exported Eclipse files. Select one or more files and hit the run button to generate a table for all selected files. The file characteristics consist of: file path, file size, file name, date of measuring, duration of measurement and which types of modalities are present in the file.

Please select one or more files of which you want to obtain the file characteristics.
