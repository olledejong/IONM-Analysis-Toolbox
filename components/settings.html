Set python src directory

This entire GUI relies on Johan Schneiders and Menno Gerbens their Pyhton Project (see the about section). If this path is not correctly configured, none of the tools will work. Make sure you select the right path.
NOTE: After configuring this once, the path will be saved in in your user-preferences. Don't worry, you can change this as often as you want.

Set default select directory

For your convenience you're able to select the default path the file / directory select window opens in. The currently configured default select path is displayed within the white box. Select and set the default select path to your liking.

Set database

The currently configured database is displayed within the white box. Select and set the the database you wish to work with.

Setup a new database

ONLY FOR NEW AND EMPTY DATABASES! The currently set database path is displayed within the white box. Click on the button to setup the database.
NOTE: Be very thoughtful when using this tool. If you (mistakenly) setup a database which is already in use and potentially full of data, you will delete all of its content or corrupt the database.

Trace selection settings

Settings that alter the user's trace selection experience. The currently configured settings are being displayed in the input fields. Alter these and hit the submit button to update the values. For now, only the chunk size setting has been implemented.
