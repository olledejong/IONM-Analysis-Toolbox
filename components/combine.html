Combine database statistics with a Graphical Analysis Data file

Using this tool, you will be able to combine the data from the configured Access database with the output of the graphical analysis data program created by Jan-Willem (Neurophysiologist at the UMCG). The 'Patient ID' argument gets used for adding the patient identifier to the output file which contains the database data as well as the output of the graphical analysis data program. The output files will have the same name as the input file, but with the addition "_COMBINED".

Please select a file you wish to combine with the Access database statistics.
