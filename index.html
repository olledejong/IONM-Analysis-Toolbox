Summarize File(s)

Gain insight into Eclipse files

python command: ionm.py summarize

Show Timing

Plot timestamps of measurements as a function of position in file

python command: ionm.py show_timing

Show EEG Availability

Generate a plot showing concurrent availability of a continuous modality and a triggered modality

python command: ionm.py show_availability

Convert File(s)

Convert an Eclipse CSV into multiple custom files: one per modality

python command: ionm.py convert

Compute Statistics

Calculate statistics for given converted file (triggered modalities) and store these in the database

python command: ionm.py compute

Extract EEG

Create separate file containing evoked potentials with the available EEG context

python command: ionm.py extract_eeg

Validate EEG/TES-MEP

Validate the signals which are located in an extracted file

python command: ionm.py validate

Combine

Combines the data from the database with the output of the graphical analysis data

python command: ionm.py combine

Classify

Classify signals of file(s) on the presence of F-waves

python command: ionm.py classify
